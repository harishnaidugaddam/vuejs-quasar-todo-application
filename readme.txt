# VueJS Quasar To-Do Application

An incredible To-Do application built using the Vue.js Quasar Framework. This web application allows users to create, manage, and organize tasks efficiently.

## Live Demo

You can access the live web app here: [VueJS Quasar To-Do Application](https://todo-by-harish.netlify.app/#/auth)

## Features

- **User Authentication**: Secure login and sign-up system.
- **Task Management**: Create, edit, and delete tasks with ease.
- **Task Status**: Mark tasks as complete or incomplete.
- **Task Prioritization**: Organize tasks based on priority levels.
- **Responsive Design**: Fully responsive and mobile-friendly UI built with VueJS and Quasar.

## Technologies Used

- **Frontend**: Vue.js, Quasar Framework, HTML5, CSS3
- **State Management**: Vuex
- **Routing**: Vue Router
- **Authentication**: Custom authentication system (local storage or API-based)
  
## Installation

To run the application locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/harishnaidugaddam/vuejs-quasar-todo-application.git
   cd vuejs-quasar-todo-application
   ```

2. **Install dependencies**:
   Make sure you have `npm` or `yarn` installed. Run the following command to install the dependencies:
   ```bash
   npm install
   ```

3. **Run the application**:
   Start the development server:
   ```bash
   npm run dev
   ```

4. **Access the app**:
   Open your browser and visit [http://localhost:8080](http://localhost:8080).

## How to Use

1. **Sign up** or **Log in** with your credentials.
2. **Create a new task** by clicking on the 'Add Task' button.
3. **Edit or delete tasks** as needed by clicking on the task item.
4. **Mark tasks as complete** or **incomplete**.
5. **Organize tasks** by priority.
